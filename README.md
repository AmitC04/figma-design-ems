## ğŸš€ Installation

### Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v16 or higher recommended)
- **npm** or **yarn** package manager
- **Git** for version control

### Quick Start

```bash
# Clone the repository
git clone https://github.com/AmitC04/figma-design-ems.git

# Navigate to project directory
cd figma-design-ems

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

## ğŸ’» Usage

### Development Mode

```bash
npm run dev
```
Runs the app in development mode with hot module replacement enabled. The page will reload automatically when you make changes.

### Production Build

```bash
npm run build
```
Builds the app for production to the `dist` folder. It correctly bundles in production mode and optimizes the build for the best performance.

### Preview Production Build

```bash
npm run preview
```
Locally preview the production build before deploying.

### Type Checking

```bash
npm run type-check
```
Run TypeScript compiler to check for type errors without emitting files.
